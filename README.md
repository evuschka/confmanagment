# Ассемблер для учебной виртуальной машины (УВМ)

## Синтаксис ассемблера
Программа на ассемблере УВМ состоит из инструкций, по одной на строку. Поддерживаются следующие команды:

- `load_const <число>`: загружает константу `<число>` на стек.  
  Диапазон: 0–32767 (15 бит)
- `read_mem <адрес>`: читает значение из памяти по адресу `<адрес>` и кладет на стек.  
  Диапазон: 0–134217727 (27 бит)
- `write_mem <смещение>`: записывает значение в память. Снимает со стека два значения: сначала значение для записи, затем базовый адрес. Записывает по адресу `базовый_адрес + смещение`.  
  Диапазон: 0–511 (9 бит)
- `neq_mem`: операция "не равно". Снимает со стека два значения: сначала второй операнд (b), затем адрес (addr). Читает из памяти по `addr` значение (a), сравнивает с b. Результат (1 если a != b, иначе 0) кладется на стек.

Комментарии начинаются с `;` и продолжаются до конца строки.  
Числа могут быть в десятичной или шестнадцатеричной системе (префикс `0x`).
